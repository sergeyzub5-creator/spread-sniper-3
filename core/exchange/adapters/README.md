# Exchange Adapters

Этот слой хранит прикладную логику бирж, отделенную от UI и от базовых классов коннекторов.

Рекомендуемая схема внутри каждой биржи:

- `account.py` — баланс, позиции, режимы аккаунта
- `market_data.py` — цены, стакан, стримы
- `trading.py` — типы ордеров, исполнение, валидации
- `<feature>.py` — feature-специфичные фасады (например `spread.py`)

Текущая реализация:

- `binance/spread.py` — пары аккаунта + snapshot bid/ask для вкладки снайпинга спреда.
